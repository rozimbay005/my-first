## 9.4 Bug Bounty — Pul ishlash

Bug Bounty — kompaniyalar o'z tizimlaridagi zaifliklarni topib berganlar uchun pul to'laydi.

### Platformalar:

| Platforma | Eng katta mukofot | Manzil |
|-----------|------------------|--------|
| **HackerOne** | $1,000,000+ | hackerone.com |
| **Bugcrowd** | $100,000+ | bugcrowd.com |
| **Intigriti** | $20,000+ | intigriti.com |
| **Synack** | Yopiq, yuqori | synack.com |
| **YesWeHack** | €50,000+ | yeswehack.com |

### Mukofot miqdorlari:

```
Critical (RCE, Auth Bypass):  $5,000 — $100,000+
High (SQLi, XSS Stored):      $500  — $10,000
Medium (IDOR, CSRF):          $100  — $1,000
Low (Info Disclosure):        $50   — $300
```

### Mashhur bug bounty topilmalari:

```
- Facebook SSRF:           $31,500
- Uber RCE:                $10,000
- Google Auth Bypass:      $36,337
- Apple Sign-In zaiflik:   $100,000
- TikTok zaiflik:          $3,860
```

### Bug Bounty boshlash uchun qadamlar:

```bash
# 1. HackerOne.com ga ro'yxatdan o'ting
# 2. "Hacktivity" bo'limini o'qing (real topilmalar)
# 3. Disclosed (ochiq) reportlarni o'qing
# 4. "Bepul" dasturlardan boshlang

# Boshlash uchun yaxshi dasturlar:
# - Yuga Labs (defi/web3)
# - Immunefi (crypto)
# - HackerOne public programs

# 5. Scope ni o'qing (nima ruxsat, nima yo'q)
# 6. Subdomain topishdan boshlang
subfinder -d target.com | httpx | tee subdomains.txt

# 7. Har bir subdomainda zaiflik qidiring
nuclei -l subdomains.txt -t exposures/  # Nuclei scanner
```

### Bug Bounty metodologiyasi:

```bash
# 1. Recon
subfinder -d target.com -o subs.txt
cat subs.txt | httpx -status-code -title > tirik_subs.txt
waybackurls target.com > urls.txt      # Tarixiy URLlar
gau target.com >> urls.txt             # Yana URLlar

# 2. Zaiflik qidirish
# XSS
cat urls.txt | gf xss | qsreplace '"><script>alert(1)</script>' | \
  httpx -match-regex "<script>alert"

# SQLi
cat urls.txt | gf sqli > sqli_params.txt
sqlmap -m sqli_params.txt --batch --dbs

# Open Redirect
cat urls.txt | gf redirect | qsreplace 'https://evil.com'

# SSRF
cat urls.txt | gf ssrf | qsreplace 'http://169.254.169.254/'

# 3. Hisobot yozish
# - Aniq tavsif
# - PoC (qadam-qadam)
# - Impact (ta'sir)
# - Tavsiya (fix)
```
